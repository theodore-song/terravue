How it works

01

Pull the data

Daily prices for ~550 names — the S&P 500, Nasdaq-100 and major ETFs — from Yahoo Finance.

02

Score every name

Four technical strategies blend into one conviction score and a BUY / HOLD / SELL call per stock.

03

Write the read

A plain-English daily note summarizes the setups — rule-based, or written by Claude when enabled.

04

Trade it

The agent sizes positions by conviction and volatility, then runs automatically every weekday after the close.

Terravue is an educational paper-trading simulation on delayed market data. It places no real orders and is not financial advice.
